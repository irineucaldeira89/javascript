Sabemos que o navegador entende apenas HTML e que se colocarmos scripts diretamente no corpo da mensagem eles serão interpretados como texto. Como resolvemos isso? Através da tag . Tudo que estiver dentro dela será interpretado como script e não texto. Correto, a tag script possibilita que tudo que estiver dentro dela seja interpretado com um script de Javascript. A tag Quando o navegador passar pela tag abaixo ele vai interprar o texto "alert("Olá mundo");" e irá executar a função alert(), que é responsável por exibir uma pop up com a mensagem Olá mundo.
